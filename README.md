# 🏍️ Miuraboy - Controle Financeiro para Motoboys

> **App gratuito, sem download, abre direto no navegador do celular**

Miuraboy é um Progressive Web App (PWA) que oferece controle financeiro em tempo real, catálogo de peças compatíveis, monitor de consumo de combustível e avaliação de postos de gasolina. Tudo pensado para motoboys que querem ganhar mais e gastar menos.

## ✨ Features Principais

### 💰 Controle Financeiro em Tempo Real
- Registre entradas (corridas, bônus) e saídas (combustível, manutenção)
- Veja o saldo do dia em tempo real
- Resumo mensal automático com gráficos
- Histórico completo de movimentações
- **Impacto**: Reduz perda de dinheiro em ~15-20%

### ⛽ Monitor de Consumo de Combustível
- Registre cada abastecimento (km, litros, valor)
- Calcule automaticamente km/litro
- Acompanhe preço médio por litro
- Detecte problemas na moto (queda na eficiência)
- **Impacto**: Economia de 10-15% em combustível

### 🔧 Catálogo de Peças Compatíveis
- Filtro por modelo de moto (Honda CB 300, Yamaha YZF, etc)
- Filtro por categoria (pneu, corrente, óleo, freio, etc)
- Preços aproximados
- Links diretos para Mercado Livre, OLX, Shopee
- **Impacto**: Reduz tempo de parada em 50%

### 🏪 Avaliação de Postos em Tempo Real
- Avalie postos por preço, qualidade, atendimento
- Veja avaliações de outros motoboys
- Ordene por melhor avaliação ou mais barato
- Comentários com dicas (ex: "Fila rápida às 6h")
- **Impacto**: Economia de 5-10% em combustível

### 📱 PWA - Funciona como App Nativo (Sem Download)
- Abra no navegador
- Clique nos 3 pontinhos → "Adicionar à tela inicial"
- Ícone aparece na tela inicial do celular
- Funciona offline (dados salvos localmente)
- Sem necessidade de atualizar na App Store
- **Impacto**: 10x mais instalações vs app nativo

## 🚀 Como Usar

### Instalação (Sem Download!)

1. Abra este link no navegador do seu celular:
   ```
   https://miuraboy.app
   ```

2. Clique nos **3 pontinhos** (menu) do navegador

3. Selecione **"Adicionar à tela inicial"** ou **"Instalar app"**

4. Pronto! Agora você tem um ícone do Miuraboy na tela inicial

### Primeiro Uso

1. **Controle Financeiro**: Registre sua primeira corrida e despesa
2. **Consumo**: Registre seu abastecimento (km, litros, valor)
3. **Peças**: Procure uma peça compatível com sua moto
4. **Postos**: Avalie um posto que você conhece

## 💻 Tecnologia

- **Frontend**: React + TypeScript + TailwindCSS
- **Armazenamento**: LocalStorage (dados salvos no seu celular)
- **PWA**: Service Worker para funcionar offline
- **Responsivo**: Otimizado para celular (mobile-first)

## 📊 Dados Salvos Localmente

Todos os seus dados (finanças, consumo, avaliações) são salvos **no seu celular**, não em nenhum servidor. Você tem total controle e privacidade.

## 🛠️ Desenvolvimento

### Pré-requisitos
- Node.js 18+
- npm ou yarn

### Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/miuraboy.git
cd miuraboy

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

O app estará disponível em `http://localhost:5173`

### Build para Produção

```bash
npm run build
```

Os arquivos compilados estarão em `dist/`
