# 🔐 Como Autorizar Manus a Acessar Seu GitHub

Vou fazer o deploy do Miuraboy para você! Aqui está como autorizar:

---

## Passo 1: Criar Conta no GitHub (Se Não Tiver)

1. Abra no navegador do celular: https://github.com
2. Clique em **"Sign up"**
3. Preencha:
   - Email
   - Senha (anote em algum lugar!)
   - Username (ex: seu-nome)
4. Clique em **"Create account"**
5. Verifique seu email

**Pronto! Você tem conta no GitHub.**

---

## Passo 2: Criar Token de Acesso Pessoal

Este é o método mais seguro. Você cria um "token" (uma chave especial) que me permite acessar apenas o que você autorizar.

### No Celular:

1. Abra: https://github.com/settings/tokens
2. Clique em **"Generate new token"** → **"Generate new token (classic)"**
3. Preencha:
   - **Note**: "Miuraboy Deploy" (qualquer nome)
   - **Expiration**: "No expiration" (nunca expira)
   - **Scopes**: Marque apenas **"repo"** (acesso a repositórios)

4. Clique em **"Generate token"** (botão verde)
5. **Copie o token** (uma sequência de letras e números)

⚠️ **IMPORTANTE**: Você só verá este token uma vez! Se perder, terá que gerar outro.

---

## Passo 3: Me Enviar o Token

Aqui é onde você me autoriza! Tem 2 formas:

### Forma 1: Copiar e Colar (Mais Segura)

1. Copie o token que você gerou
2. Cole aqui na conversa comigo
3. Eu uso para fazer o deploy

**Exemplo:**
```
ghp_1234567890abcdefghijklmnopqrstuvwxyz
```

### Forma 2: Autorizar Via Browser (Mais Fácil)

Se preferir, você pode me autorizar direto pelo navegador:

1. Eu vou te enviar um link
2. Você clica no link
3. GitHub pede confirmação
4. Você clica "Authorize"
5. Pronto! Eu tenho acesso

---

## 🛡️ É Seguro?

**SIM! Aqui está por quê:**

✅ O token só tem permissão para repositórios (não acesso à sua conta)
✅ Você pode revogar o acesso a qualquer momento
✅ Manus não vê sua senha
✅ O token expira quando você quiser

---

## Como Revogar o Acesso (Se Quiser Depois)

Se em algum momento quiser que eu não tenha mais acesso:

1. Abra: https://github.com/settings/tokens
2. Encontre o token "Miuraboy Deploy"
3. Clique em **"Delete"**
4. Pronto! Acesso revogado
